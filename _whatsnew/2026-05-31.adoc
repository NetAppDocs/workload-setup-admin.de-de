=== Unterstützung für AWS GovCloud (US)-Konten

Sie können jetzt AWS GovCloud (US)-Konten in Workload Factory erstellen. Dieser Kontotyp verwendet ausschließlich AWS US-GovCloud-Endpunkte und -Regionen für Einrichtung und Betrieb. Anmeldeinformationen und Geheimnisse verbleiben in GovCloud. GovCloud (US)-Regionen werden unterstützt.

link:https://docs.netapp.com/us-en/workload-setup-admin/sign-up-saas.html["Erstellen Sie ein AWS GovCloud (US)-Konto in Workload Factory"]
