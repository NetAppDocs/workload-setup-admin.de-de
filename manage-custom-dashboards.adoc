= Benutzerdefinierte Dashboards verwalten
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Verwalten Sie Ihre benutzerdefinierten Dashboards im Dashboard-Katalog. Von dort aus lassen sich Details anzeigen, Daten aktualisieren, Dashboard-Einstellungen bearbeiten oder Dashboards entfernen, die nicht mehr benötigt werden.

.Über diese Aufgabe
Alle Benutzer Ihres Workload Factory Kontos haben Zugriff auf alle benutzerdefinierten Dashboards. Jedes Dashboard zeigt grundlegende Details wie den Namen des Erstellers, das Erstellungsdatum und ob es derzeit von jemandem bearbeitet wird.

Benutzerdefinierte Dashboards sind schreibgeschützt. Sie können weder Ihre Speicher- noch Ihre Workload-Einstellungen ändern. Das Dashboard selbst kann bearbeitet werden, jedoch nicht die von ihm überwachten Ressourcen.

.Bevor Sie beginnen
Sie müssen mindestens ein benutzerdefiniertes Dashboard in Ihrem Workload Factory Konto haben.



== Auf den Dashboard Katalog zugreifen

.Schritte
. Melden Sie sich mit einem der link:console-experiences.html["Konsolenerfahrungen"^] an.
. Im Menü *Workloads Overview* und anschließend *Custom dashboards* auswählen.
+
Der Dashboard Katalog zeigt alle benutzerdefinierten Dashboards in Ihrem Workload Factory Konto an.

. Die Dashboard-Liste wird angezeigt.




== Dashboard-Daten aktualisieren

Dashboard-Daten werden beim Öffnen eines Dashboards geladen. Eine Aktualisierung zeigt die neuesten Informationen an.

Das Dashboard lädt Daten aus Ihren Workloads neu. Die Zeitangabe „Letzte Aktualisierung“ ändert sich und zeigt an, wann die Daten aktualisiert wurden.



== Ein Dashboard bearbeiten

Ein Dashboard kann bearbeitet werden, um anzuzeigen, welche Metriken und Diagramme dargestellt werden.

.Schritte
. Auf der Seite *Benutzerdefinierte Dashboards* das Dashboard suchen, das bearbeitet werden soll.
. Im Menü *Aktionen* für dieses Dashboard die Option *Dashboard bearbeiten* auswählen.
+
Wenn ein anderer Benutzer das Dashboard bearbeitet, erscheint eine Meldung wie „Wird gerade von [Benutzername] bearbeitet“. Es kann abgewartet oder eine Abstimmung mit dieser Person vorgenommen werden.

. Im KI-Dialog die gewünschten Änderungen beschreiben.
+
Dieser Vorgang ähnelt dem Erstellen eines neuen Dashboards. Die Workloads, Metriken oder Zeiträume, die das Dashboard erfasst, können geändert werden.

. Die Änderungen überprüfen und **Speichern** auswählen.


.Ergebnis
Das Dashboard wird mit Ihren Änderungen aktualisiert. Das Datum der letzten Änderung wird ebenfalls aktualisiert.



== Ein Dashboard löschen

Dashboards, die nicht mehr benötigt werden, können gelöscht werden. Dieser Vorgang ist endgültig und kann nicht rückgängig gemacht werden.

.Schritte
. Auf der Seite *Benutzerdefinierte Dashboards* das Dashboard suchen, das entfernt werden soll.
. Im Dropdown-Menü *Aktionen* für dieses Dashboard die Option *Löschen* auswählen.
. Bestätigen Sie die Löschung.


.Ergebnis
Das Dashboard wird dauerhaft aus Ihrem Workload Factory-Konto entfernt und alle Benutzer verlieren den Zugriff darauf.
